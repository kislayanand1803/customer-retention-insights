# 📊 Churn Probability & Customer Segmentation

## 🔎 Overview
This project explores customer churn prediction and segmentation using machine learning techniques. Built in **Google Colab**, it combines predictive modeling with clustering methods to provide actionable insights for customer retention strategies.

## 📂 Dataset
- **Source:** (https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset)
- **Features:** Customer demographics, transaction history, service usage, etc.
- **Target:** Churn probability

## ⚙️ Methodology
1. **Data Preprocessing**
   - Cleaning, handling missing values, encoding categorical variables
2. **Exploratory Data Analysis (EDA)**
   - Visualizations of churn distribution, customer segments
3. **Modeling**
   - Classification models (e.g., Logistic Regression, Random Forest, XGBoost)
   - Evaluation metrics: Accuracy, Precision, Recall, F1-score, ROC-AUC
4. **Segmentation**
   - Clustering techniques (e.g., K-Means, Hierarchical Clustering)
   - Profiling customer groups for targeted strategies

## 📈 Results
- Churn prediction model performance summary
- Key customer segments identified
- Business insights for retention and marketing
